# MySQL Practice Project

## Overview
This repository contains MySQL queries and database concepts practiced for learning and placement preparation.

## Topics Covered
- DDL Commands (CREATE, ALTER, DROP, TRUNCATE)
- DML Commands (INSERT, UPDATE, DELETE)
- DQL Commands (SELECT)
- Constraints (PRIMARY KEY, FOREIGN KEY, UNIQUE, CHECK)
- Joins (INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL JOIN, SELF JOIN)
- Aggregate Functions
- GROUP BY and HAVING
- Subqueries
- Views
- Stored Procedures
- Triggers

## Files
- `query1.sql` - Contains SQL queries and examples.

## Technologies Used
- MySQL
- MySQL Workbench

## Purpose
This project is created for practicing SQL concepts, database management, and placement preparation.
